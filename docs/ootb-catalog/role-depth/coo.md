# COO

## 1. Role summary

Operations: drives execution across functions, owns operational rhythm, removes blockers, coordinates cross-functional initiatives. COO Twin synthesizes execution-rollup briefs + cross-functional blocker maps + vendor / renewal reviews + weekly execution reviews. Never employee scoring. Never surveillance framing.

## 2. Common titles

COO · Chief of Staff to CEO · VP Operations.

## 3. Likely reports to

CEO.

## 4. Possible reports to

Board (in tech-light orgs where COO is #2).

## 5. Likely direct reports

VP Operations · Heads of Customer / Sales / People in some structures · Project Manager / PMO.

## 6. Possible direct reports

Office Manager · Chief of Staff team.

## 7. Dotted-line relationships

CTO · CFO · CHRO · CRO · Head of Product.

## 8. Cross-functional partners

All executives — COO is the connective tissue.

## 9. External collaborators

Vendors · partners · external service providers · landlord / facilities (in some orgs).

## 10. Core responsibilities

Operational rhythm · cross-functional execution · blocker escalation · OKR tracking · vendor renewals.

## 11. Common decisions

Resource reallocation · operational priority calls · vendor approvals (within spend).

## 12. Common meetings

Weekly leadership · operational review · quarterly OKRs · vendor reviews.

## 13. Common documents / artifacts

OKR dashboard · execution review · vendor list · cross-functional priority list.

## 14. Common metrics / KPIs

Execution velocity · OKR progress · burn vs progress. Never employee scoring.

## 15. Common tools

Google Calendar / Outlook · Slack / Teams · Jira / Linear · Notion / Confluence · DocuSign · Coupa / NetSuite (vendors).

## 16. Common workflows

- Weekly Execution Review
- Cross-Functional Blocker Escalation
- Vendor Renewal Review
- Quarterly OKR Review

## 17. Approval authority

Vendor spend within ceiling. Operational reallocation within authority.

## 18. Approval dependencies

Vendor renewals above threshold → CFO · headcount changes → CHRO + CEO · contract terms → GC.

## 19. Delegated authority

Calendar / email delegated to EA.

## 20. Never-default permissions

Employee scoring · manager surveillance framing · cross-tenant signals · regulatory disclosure without GC.

## 21. Digital Twin day-one capabilities

Weekly execution rollup · cross-functional blocker map · vendor renewal brief · OKR progress synthesis.

## 22. First-week aha moments

1. **Weekly Execution Snapshot** — team-level OKR progress + risks.
2. **Cross-Functional Blocker Map** — who's stuck on whom.
3. **Vendor Renewal Brief** — upcoming renewals + spend + risk.
4. **Quarterly OKR Rollup** — synthesized progress.
5. **Operational Health Brief** — cadence + meetings + decisions.

## 23. Safe fallback without connectors

Execution rollup template · blocker map template · vendor renewal template.

## 24. Connector implications

Slack · Project Tracker (Jira / Linear) · DocuSign · NetSuite (vendor summaries).

## 25. DMW / Memory Wallet scope notes

Self-scoped operational context · cross-functional execution scope · vendor context · OKR scope. Forbidden: employee-level performance signals; protected-attribute inference.

## 26. Governed context envelope notes

- `object_type`: OPERATIONS_EXECUTIVE_OPERATING_MODEL
- `policy_purpose`: GOVERNED_OPERATIONAL_STEWARDSHIP
- `permission_defaults`: READ_FIRST_CROSS_FUNCTIONAL, VENDOR_APPROVAL_WITHIN_SPEND, NEVER_EMPLOYEE_SCORING
- `sensitivity_level`: HIGH
- `forbidden_consumers`: EMPLOYEE_SCORING_TOOL · MANAGER_SURVEILLANCE_TOOL

## 27. Collaboration map

**Upward**: CEO. **Downward**: VP Ops + PMs. **Peer**: All C-suite. **Cross-functional**: every dept. **External**: vendors, partners. **Approval path**: vendor spend → CFO; HR changes → CHRO; legal terms → GC. **Escalation**: cross-functional crisis → CEO + GC.

## 28. Risks and guardrails

Employee-level scoring · vendor over-commit · cross-tenant visibility · audit suppression.

## 29. Industry / company-size variants

- **Startup**: COO + Chief of Staff often combined.
- **Mid-market / Enterprise**: dedicated COO + VP Ops + PMO.
- **Regulated**: vendor renewals tied to compliance audit.
